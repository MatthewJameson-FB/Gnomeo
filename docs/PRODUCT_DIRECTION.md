# Product Direction

## Current direction
Gnomeo is a private AI-assisted ad-spend review layer.

It focuses on:
- what deserves attention
- calm, concise analyst-style summaries
- helping people review ads faster and with less noise

It is not:
- a dashboard
- an ad manager replacement
- a BI tool
- an AI optimisation engine

## Core promise
Less confusion, not more data.

Gnomeo should help users understand:
- what changed
- what looks unusual
- what to review first
- what may need a human decision

## Target users
Early audience:
- founder-led businesses
- small teams without a dedicated marketing analyst
- solo marketers
- small ecommerce operators
- fractional marketers
- small agencies later

Core pain:
- they do not need more dashboards
- they need to know what deserves attention first

## Surfaces
### Chrome extension
Primary future surface.
- "Review this page"
- local extraction of visible campaign table data
- user-triggered only
- no screenshots by default
- no background monitoring
- no platform APIs yet
- no OAuth yet
- save useful spot checks to the private workspace when the user chooses

### Portal
Secondary but important.
- private review file
- saved reviews
- review history
- recurring memory
- deeper CSV paste/upload
- comparison against previous reviews
- receives token-scoped extension saves for the same workspace
- receives Chrome visible-table reports that are not full account audits

### Public site
- explains the product
- drives request access / beta interest
- positions Gnomeo as a review layer + private review file

## Competitive posture
Do not compete on:
- feature count
- dashboards
- attribution accuracy
- auto-optimisation
- campaign building

Compete on:
- independent cross-platform review
- low-noise prioritization
- recurring memory
- calm "what deserves attention" summaries
- less confusion, not more data
- a simple loop: Review in Chrome → Save to workspace → Compare over time
