# Pricing Model

## Current pricing direction
Billing is not implemented yet.

CTAs should remain:
- Request access
- beta interest

Do not imply live checkout until payment is actually added.

## Free
- 3 page reviews per month
- visible table spot-check
- top finding
- key signals
- limited or no persistent memory

## Starter — £15/month
- 50 reviews per month
- saved review history
- recurring review memory
- CSV paste/upload
- one workspace/account
- comparison against previous reviews
- Chrome visible-table reports

## Pro — £39/month
- 250 reviews per month
- multiple workspaces/accounts
- cross-platform review memory
- export/share report later
- deeper visible-table comparison

## Agency
- later only
- do not build now

## Pricing guardrails
- keep pricing simple
- do not turn the product into a dashboard bundle
- do not anchor on optimisation promises
- sell memory, review focus, and calm prioritization
